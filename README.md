# Sunset

Sunri体重秤的翻版App
# 版本号
- Flutter ^2.8.0
- Dart ^2.15.0
- Andrio Studio ^4.1
- Kotlin ^1.4.32
- Java ^11.0.12

# 说明

Sunset数据来源 [sunset-server](https://github.com/dlongs49/sunset-server)

功能：

- 登录，注册，密码相关，注销

- 发表动态，评论
- 动态，评论点赞
- 动态，评论分页
- 主题换肤
- 二维码扫描
- ......
